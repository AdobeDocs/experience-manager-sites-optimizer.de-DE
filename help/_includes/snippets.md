# Snippets

## Genehmigung anfordern {#auto-optimize-request-approval}

Wenn Sie die Optimierung nicht bereitstellen können, können Sie die Genehmigung bei den Admins Ihrer AEM-Site anfordern. Wenn Sie **Genehmigung anfordern** auswählen, wird eine E-Mail mit den Details der Optimierung an die oder den Admin der AEM-Site gesendet. Die bzw. der Admin kann dann die Optimierung genehmigen oder ablehnen.

## Optimierung bereitstellen {#auto-optimize-deploy-optimization-slack}

Das Team von Adobe hilft Ihnen gerne. Wenn Sie Support bei technischen Optimierungen benötigen, treten Sie dem Slack-Kanal von Adobe bei und vernetzen Sie sich direkt mit Adobe-Fachleuten.
